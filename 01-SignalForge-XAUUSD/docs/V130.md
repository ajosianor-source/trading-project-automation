# SignalForgeFX v1.30

Version 1.30 is installed alongside v1.20. It does not replace or reconfigure
the v1.20 instance currently attached to the XAUUSD chart.

## Safety and execution changes

- Position size is calculated from the executable bid/ask rather than the
  previous candle close.
- Stop and target are rebuilt around the executable price using current ATR.
- Fixed-point and ATR-relative spread gates must both pass.
- Broker session, symbol direction, stop/freeze distance, and `OrderCheck`
  validation run before every request.
- The post-fill monetary risk is recalculated; a material risk overshoot closes
  the position and permanently halts the EA instance.
- Daily loss, total drawdown, consecutive-loss, and trades-per-day breakers
  block new entries.
- An emergency input closes positions owned by the v1.30 magic number and
  permanently halts that instance.
- Entry, rejection, skip, and exit events are written to
  `Common/Files/SignalForge/trades.csv`.
- Server-confirmed exits produce phone notifications with P/L and R.

## Required regression test

Do not attach v1.30 to the live demo chart yet.

1. Open MT5 Strategy Tester.
2. Select `SignalForgeFX_v130/SignalForgeFX_v130`.
3. Choose XAUUSD, H1, and **Every tick based on real ticks**.
4. Load `SignalForge_v130_demo.set`.
5. Set 2025.05.01 through 2026.06.26.
6. Enable random execution delay and run the test.
7. Compare the report with the v1.20 held-out baseline:
   - 70 trades;
   - $1,065.21 net profit on $10,000;
   - profit factor 1.63;
   - maximum equity drawdown 2.48%.

The results do not need to be identical because live-price stops, preflight
checks, and circuit breakers intentionally change execution. Reject v1.30 if it
introduces unexplained errors, no trades, materially larger drawdown, or a
negative held-out result.

## Safe migration after the test passes

1. Save the v1.30 tester report.
2. Keep v1.20 attached until the H1 candle has closed and no position is open.
3. Remove v1.20 from the XAUUSD chart.
4. Attach `SignalForgeFX_v130`.
5. Load `SignalForge_v130_demo.set`.
6. Confirm the account is FTMO Demo and phone notifications work.
7. Verify the chart says `SIGNALFORGE FX 1.30`.

Never run v1.20 and v1.30 on the same symbol simultaneously.
