# Security policy

Report suspected vulnerabilities privately through GitHub Security Advisories. Do not
include PHI, credentials, patient screenshots, or production log extracts. The security
owner is Joy Abu.

Critical reports are triaged within 4 hours, high within 1 business day, and other reports
within 3 business days. Supported releases are the current production release and the
immediately preceding release.

Production changes require peer review, passing security and compliance gates, signed
artifacts, an approved change record, and a tested rollback. Emergency changes must be
retrospectively reviewed within one business day.
